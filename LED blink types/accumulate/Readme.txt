Simple implementation of accumulation using 5 LEDs

Here 'X' representss LED ON
while 'O' represents LED OFF

O O O O X -> 1
O O O X O -> 2
O O X O O -> 4
O X O O O -> 8
X O O O O -> 16
X O O O X -> 17
X O O X O -> 18
X O X O O -> 20
X X O O O -> 24
X X O O X -> 25
X X O X O -> 26
X X X O O -> 28
X X X O X -> 29
X X X X O -> 30
X X X X X -> 31
REPEAT
